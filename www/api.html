SpaceBoteque API

Упрощённая надстройка над Launch Library 2 API о предстоящих пусках научных миссий по исследованиям Солнечной Системы, Галактики, Вселенной

Методы API
  • launchStatuses: статусы пусков
  • missionTypes: типы миссий
  • launches: предстоящие пуски
  • mission: запускаемая миссия
  • agency: космическое агентство
  • location: космодром

Краткий глоссарий

  • launch - пуск ракеты-носителя с некой полезной нагрузкой (mission)
  • mission - миссия (полезная нагрузка) пуска (launch)
  • launchStatus - текущий статус пуска
  • missionType - тип миссии
  • agency - космическое агентство
  • location - космодром
  • pad - стартовая площадка

Описание API

  • URL: http://sbq.driven.ru/api.php
  • API работает по протоколу JSON-RPC 2.0 ⇐ это ссылка на спецификацию
  • Все запросы посылать на указанный адрес POST'ом в виде json-строки, сформированной соответственно спецификации. Значения аргументов являются регистрозависимыми! В настоящее время API принимает только по одному запросу!
  • Все аргументы, передаваемые в params являются обязательными; возможные необязательные аргументы обговорены отдельно
  • Все ответы приходят аналогично соответственно спецификации. Успешность / неуспешность выполнения запроса определяется наличием в ответе значения с ключом result или error соответственно. При успешном выполнении метода result в большинстве случаев представляет собой массив данных (см. описания методов), если не указано иное. В случае невыполнения метода error содержит в себе два поля с ключами code (код ошибки) и message (текстовое описание)
  • Сервис живёт по времени UTC

launchStatuses: статусы пусков

Передаваемые аргументы: нет

{"jsonrpc":"2.0","id":1777494943128,"method":"launchStatuses"}

В result передаётся массив данных, каждый элемент соответствует одному статусу пуска:

  • int id - id статуса пуска
  • string name - наименование
  • string abbrev - сокращённое наименование
  • string description - описание
{"jsonrpc":"2.0","id":1777494943128,"result":[{"id":1,"name":"Go for Launch","abbrev":"Go","description":"..."},...]}

launches: предстоящие пуски

Список предстоящих пусков, отсортированный по дате и времени от ближайшего и дальше

Передаваемые аргументы:

  • int offset - индекс первой записи; необязательный аргумент, значение по умолчанию равно 0
  • int limit - кол-во записей на запрос; необязательный аргумент, значение по умолчанию равно 10

{"jsonrpc":"2.0","id":1777731925688,"method":"launches"}
{"jsonrpc":"2.0","id":1777742020322,"method":"launches","params":{"offset":0,"limit":6}}

Получаемый массив данных в result:

  • int count - общее количество имеющихся записей о предстоящих пусках
  • array launches - массив записей о предстоящих пусках, каждая запись соответствует одному пуску:
    • string uuid - uuid пуска
    • string name - наименование
    • int status - статус пуска (см. статусы пусков)
    • array location - космодром:
      • int id - id космодрома
      • string name - наименование
    • array pad - стартовая площадка:
      • int id - id стартовой площадки
      • string name - наименование
    • int net - дата и время пуска (значение передаётся в unix time)
    • array window - пусковое окно (значения передаются в unix time):
      • int start - открытие
      • int end - закрытие
    • int updated - дата и время последнего обновления (значение передаётся в unix time)

{"jsonrpc":"2.0","id":1777742020322,"result":{"count":34,"launches":[{"uuid":"b31313da-ee5d-42b2-9a27-20cc18105276","name":"Vega-C | Solar wind Magnetosphere Ionosphere Link Explorer (SMILE)","status":1,"location":{"id":13,"name":"Guiana Space Centre, French Guiana"},"pad":{"id":34,"name":"Ariane Launch Area 1 (ELV)"},"net":1779162720,"window":{"start":1779162720,"end":1779162720},"updated":1776966775},...]}

mission: запускаемая миссия

Передаваемые аргументы:

  • string uuid - UUID пуска

{"jsonrpc":"2.0","id":1777801786645,"method":"mission","params":{"uuid":"b31313da-ee5d-42b2-9a27-20cc18105276"}}

Получаемый массив данных в result:

  • string name - наименование
  • array type - тип миссии:
    • int id - id типа миссии
    • string name - наименование
  • string description - описание
  • array agencies - агентства, к которым относится миссия; массив записей, каждая из которых относится к отдельному агентству
    • int id - id агентства
    • string name - наименование
    • string abbrev - аббревиатура
    • string description - описание

{"jsonrpc":"2.0","id":1777801786645,"result":{"name":"Solar wind Magnetosphere Ionosphere Link Explorer (SMILE)","type":{"id":3,"name":"Astrophysics"},"description":"...","agencies":[{"id":27,"name":"European Space Agency","abbrev":"ESA","description":""},...]}}

location: космодром

Передаваемые аргументы:

  • int id - id космодрома

{"jsonrpc":"2.0","id":1777825399617,"method":"location","params":{"id":13}}

Получаемый массив данных в result:

  • string name - наименование
  • string description - описание
  • float latitude - широта
  • float longitude - долгота

    • string description - описание
    • float latitude - широта
    • float longitude - долгота

{"jsonrpc":"2.0","id":1777825399617,"result":{"name":"Guiana Space Centre, French Guiana","description":"...","latitude":5.169,"longitude":-52.6903,"pads":[{"name":"Ariane Launch Area 1 (ELV)","description":"...","latitude":5.236,"longitude":-52.775},...]}}

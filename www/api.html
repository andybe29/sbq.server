Краткий глоссарий

  • launch - пуск ракеты-носителя с некой полезной нагрузкой (mission)
  • mission - миссия (полезная нагрузка) пуска (launch)
  • launchStatus - текущий статус пуска
  • missionType - тип миссии
  • orbit - орбита, на которую производится пуск миссии
  • agency - космическое агентство
  • location - космодром
  • pad - стартовая площадка

Описание API

  • URL: http://sbq.driven.ru/api.php
  • API работает по протоколу JSON-RPC 2.0 ⇐ это ссылка на спецификацию
  • Все запросы посылать на указанный адрес POST'ом в виде json-строки, сформированной соответственно спецификации. Значения аргументов являются регистрозависимыми! В настоящее время API принимает только по одному запросу!
  • Все аргументы, передаваемые в params являются обязательными; возможные необязательные аргументы обговорены отдельно
  • Все ответы приходят аналогично соответственно спецификации. Успешность / неуспешность выполнения запроса определяется наличием в ответе значения с ключом result или error соответственно. При успешном выполнении метода result в большинстве случаев представляет собой массив данных (см. описания методов), если не указано иное. В случае невыполнения метода error содержит в себе два поля с ключами code (код ошибки) и message (текстовое описание)
  • Сервис живёт по времени UTC

launchStatuses: список статусов пусков

Получение списка статусов пусков

Передаваемые аргументы: нет

{"jsonrpc":"2.0","id":1777494943128,"method":"launchStatuses"}

В result передаётся массив данных, каждый элемент соответствует одному статусу пуска:

  • int id - id статуса пуска
  • string name - наименование
  • string abbrev - сокращённое наименование
  • string description - описание
{"jsonrpc":"2.0","id":1777494943128,"result":[{"id":1,"name":"Go for Launch","abbrev":"Go","description":"..."},...]}
